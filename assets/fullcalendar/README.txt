[필수] 로컬 FullCalendar 파일 추가 방법
1) 아래 4개 파일을 FullCalendar 공식 배포판에서 다운로드
   - main.min.js
   - main.min.css
   - daygrid.min.js
   - daygrid.min.css
2) 이 ZIP 안의 경로에 그대로 넣기:
   /assets/fullcalendar/main.min.js
   /assets/fullcalendar/main.min.css
   /assets/fullcalendar/daygrid.min.js
   /assets/fullcalendar/daygrid.min.css
3) GitHub Pages에 함께 업로드
4) Ctrl+Shift+R 강력 새로고침
